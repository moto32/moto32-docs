## What Your Support Enables

### Development Costs

Your contributions help cover:

:prose-ul{icon="i-lucide-banknote"}
- **Prototype PCBs**: Testing new features and improvements
- **Components**: Parts for development and testing
- **Test equipment**: Tools for validation and debugging
- **Software licenses**: CAD tools, development software
- **Hosting**: Documentation website and file storage

### Time Investment

The project requires significant time for:

:prose-ul{icon="i-lucide-clock"}
- Circuit design and optimization
- Firmware development
- Writing documentation
- Testing and validation
- Community support
- Bug fixes and updates

### Future Features

Funding helps develop exciting new features:

:prose-ul{icon="i-lucide-sparkles"}
- **Bluetooth smartphone app** for wireless configuration
- **CAN bus integration** for modern motorcycles
- **Advanced sensors** (IMU, temperature monitoring)
- **Pre-assembled units** for plug-and-play installation
- **Injection-molded enclosures** for professional appearance
- **Video tutorials** and detailed guides

## Transparency

As an open-source project, we believe in transparency:

### How Funds Are Used

:prose-ul{icon="i-lucide-pie-chart"}
- **60%** Hardware development (PCBs, components, testing)
- **20%** Software development (tools, hosting)
- **10%** Documentation (hosting, domain, images)
- **10%** Community (giveaways, support)

### Financial Updates

We'll share regular updates on:
- Development progress
- How funds were used
- Upcoming projects and goals
- Community milestones
